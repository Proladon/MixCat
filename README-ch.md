## Overview

MixCat 是一個由 [Proladon](https://github.com/Proladon) 所構想出的一個 discord bot 架構，是由最基本的 discord.js bot 結合混用了一些前端生態套件與工具，像是 `vite(前端打包工具)`、`vue(前端框架)`、`pinia(前端全域狀態管理套件)` 而成的一個 discord bot 架構。
